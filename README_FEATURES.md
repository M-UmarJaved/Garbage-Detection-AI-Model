## Quick Start

### Step 1 — Install dependencies
```bash
py -m pip install -r requirements_new.txt
```

### Step 2 — Run the main detection system
```bash
# Activate your virtual environment first (if using env2)
env2\Scripts\activate

# Then run the main system (starts ALL features automatically)
py fina-v2.py
```
This single command starts:
- ✅ YOLO real-time detection
- ✅ SQLite logging (creates `garbage.db`)
- ✅ Voice feedback (background thread)
- ✅ Bin fill HTTP server on port **5050**

### Step 3 — Open the Live Dashboard (in a separate terminal)
```bash
streamlit run dashboard.py
```
Then open your browser to: **http://localhost:8501**

### Step 4 — Generate a PDF Report (anytime)
```bash
# Daily report
py report.py

# Weekly report
py report.py --period weekly

# Monthly report
py report.py --period monthly
```

## Bin Fill Monitoring — ESP32-WROOM Setup

The ESP32-WROOM ultrasonic sensor firmware should send HTTP GET requests to:

```
http://<YOUR_LAPTOP_IP>:5050/fill?plastic=82&paper=45
```

**Example Arduino snippet** (add to your ESP32-WROOM sketch):
```cpp
#include <HTTPClient.h>

void sendFillLevel(int plasticPct, int paperPct) {
    HTTPClient http;
    String url = "http://192.168.4.2:5050/fill?plastic=" 
                 + String(plasticPct) 
                 + "&paper=" + String(paperPct);
    http.begin(url);
    int code = http.GET();
    http.end();
}

// Call this in loop() every 30 seconds
// sendFillLevel(plasticSensorReading, paperSensorReading);
```

Find your laptop IP with: `ipconfig` → look for IPv4 address on the ESP32's WiFi network.

## Database Structure (`garbage.db`)

```sql
-- Every YOLO detection event
CREATE TABLE detections (
    id            INTEGER PRIMARY KEY AUTOINCREMENT,
    timestamp     TEXT,     -- "2026-06-03 10:30:21"
    class         TEXT,     -- "Small_Bottle" or "Crushed_Paper"
    confidence    REAL,     -- 0.0 to 1.0
    fps           REAL,     -- frames per second at time of detection
    inference_ms  REAL,     -- YOLO inference time in milliseconds
    direction     TEXT,     -- "TURN_LEFT", "CENTER", etc.
    center_x      INTEGER,  -- pixel X of detected object center
    center_y      INTEGER   -- pixel Y of detected object center
);

-- Bin fill level readings from ESP32-WROOM
CREATE TABLE bins (
    id          INTEGER PRIMARY KEY AUTOINCREMENT,
    timestamp   TEXT,
    plastic_pct REAL,   -- 0-100%
    paper_pct   REAL    -- 0-100%
);

-- Each run of fina-v2.py
CREATE TABLE sessions (
    id               INTEGER PRIMARY KEY AUTOINCREMENT,
    start_time       TEXT,
    end_time         TEXT,
    total_detections INTEGER
);
```

Query your database anytime:
```bash
py -c "import logger; logger.init(); print(logger.get_summary())"
```

## Dashboard Features

When you run `streamlit run dashboard.py` you get:

1. **🎯 KPI Cards** — total detections, avg confidence, avg FPS, session count
2. **📈 Timeline Chart** — detections over time (per minute)
3. **🥧 Donut Chart** — bottle vs paper class distribution
4. **🗑️ Bin Fill Gauges** — real-time plastic & paper percentage gauges
5. **📊 Confidence Histogram** — distribution of confidence scores by class
6. **⚡ FPS Chart** — frame rate history
7. **📋 Raw Log Table** — last 50 detections with all fields
8. **⚠️ Alert Banners** — auto warning if any bin exceeds 85%
9. **🔄 Auto-refresh** — configurable 3–60 second refresh interval
